# Flomo Sync Chrome Extension

一个简单易用的 Chrome 扩展，帮助你在浏览网页时快速将内容同步到 Flomo。

## 功能特点

- 一键同步内容到 Flomo
- 自动获取当前页面的标题和链接
- 支持添加原文摘要和个人感想
- 智能写作反馈和动态按钮效果
- 内置 AI 总结功能（基于智谱 GLM-4-flash 模型）
- 现代化的响应式侧边栏界面
- 支持快捷键操作（Alt+F）

## 界面特色

- 平滑的动画过渡效果
- 渐变按钮和动态反馈
- 全响应式布局设计
  - 自适应屏幕尺寸的侧边栏宽度
  - 智能调整的输入框高度
  - 适配不同设备的字体大小
- 优雅的输入框交互
  - 实时写作反馈提示
  - 字数进度动态显示
  - 提交按钮状态动画
- 精心设计的配色方案
- 合理的空间布局

## 安装方法

1. 下载本项目代码
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 使用说明

### 首次使用配置

1. 点击插件图标右键 -> 选择"选项"，或在插件侧边栏中点击设置图标（⚙️）
2. 在设置页面中填入你的 Flomo Webhook URL
   - Webhook URL 可以在 Flomo 的 API 设置中获取
3. 可选：自定义默认标签（默认为 #Chrome阅读笔记）

### 日常使用

1. **打开侧边栏**
   - 方式一：点击浏览器工具栏中的插件图标
   - 方式二：使用快捷键 `Alt+F`

2. **同步内容**
   - 标题和链接会自动填充
   - 在"原文摘要"框中粘贴想要保存的原文内容
   - 点击"AI总结"按钮可以智能提取文章要点
   - 在"个人感想"框中输入你的想法
     - 写作过程中会有友好的反馈提示
     - 按钮会随着输入展现生动的动画效果
   - 点击"提交到 Flomo"按钮完成同步

### 写作反馈

为了鼓励更好的写作体验，扩展提供了智能的反馈机制：

- 写作进度反馈（个人感想）：
  - 0-15字：显示"写点想法..."
  - 15-30字：显示"继续写..."
  - 30-50字：显示"再来点！"
  - 50字以上：显示" 太棒了"

- 按钮动态效果：
  - 开始输入时会有轻微的跳动效果
  - 达到50字后会展现柔和的呼吸动画
  - 所有动效都经过精心设计，既有趣又不会干扰写作

### AI 总结功能

- 基于智谱 GLM-4-flash 模型
- 自动提取文章核心观点和关键信息
- 支持中英文文章，统一输出中文总结
- 总结内容结构化：
  - 核心观点（1-2条）
  - 关键信息（2-3条）
  - 独特见解（1条）
  - 行动建议（如有）

### 响应式设计

- 侧边栏宽度：
  - 最小 280px（保证内容可读）
  - 最大 380px（避免占用过多空间）
  - 自动适应屏幕宽度（25vw）

- 输入框高度：
  - 原文摘要：150px - 300px（25vh）
  - 个人感想：80px - 100px（15vh）
  - 自动适应屏幕高度

- 界面元素：
  - 字体大小自适应（vw/vh 单位）
  - 间距智能调整
  - 图标尺寸响应式变化

### 快捷键设置

1. **默认快捷键**
   - 打开侧边栏：`Alt+F`

2. **自定义快捷键**
   - 在浏览器地址栏输入：`chrome://extensions/shortcuts`
   - 找到"善思flomo"扩展
   - 点击现有快捷键，输入你想要的组合键
   - 支持的组合键包括：
     - Ctrl/Command + [字母/数字]
     - Alt + [字母/数字]
     - Shift + [字母/数字]
     - 或者它们的组合

3. **快捷键设置入口**
   - 方式一：在扩展设置页面点击"查看修改方法"按钮
   - 方式二：直接访问 `chrome://extensions/shortcuts`

## 隐私说明

- 插件仅在您主动点击提交时才会发送数据到 Flomo
- AI 总结功能使用智谱 AI 的服务，会发送文本内容到其 API
- 不会收集任何其他个人信息
