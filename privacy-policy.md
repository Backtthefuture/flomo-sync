# Flomo Sync 隐私权政策

最后更新日期：2024-12-11

## 数据收集和使用

Flomo Sync 扩展程序重视您的隐私。我们仅收集以下必要信息：

1. **页面内容**：仅当您主动使用扩展功能时，我们会访问当前页面的标题、URL 和您选中的内容。
2. **配置信息**：您的 Flomo API 配置和个性化设置（如默认标签）会存储在浏览器本地。

## 数据存储

- 所有数据仅存储在您的本地浏览器中
- 我们不会将任何数据上传到我们的服务器
- 仅在您主动同步时，选中的内容才会发送到 Flomo 服务器

## 数据共享

- 我们不会与任何第三方共享您的数据
- 内容仅在您主动操作时同步到您的 Flomo 账户

## 安全措施

- 所有与 Flomo API 的通信都使用 HTTPS 加密
- 您的 API 密钥仅存储在本地，不会传输到其他地方

## 联系我们

如果您对隐私政策有任何疑问，请通过以下方式联系我们：

- GitHub Issues: [提交问题](https://github.com/yourusername/flomo-sync/issues)

## 政策更新

我们可能会不时更新本隐私政策。任何重大变更都会通过扩展更新时通知您。
